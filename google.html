Google Coding Interview Preparation

1. Data Structures and Algorithms

  • LeetCode Problem Sets - Practice DSA problems.
  • GeeksforGeeks - Data Structures - In-depth guides and examples.
  • Coding Ninjas - Guided Paths - Structured learning paths for algorithms.
  • HackerRank - Algorithms - Learn and practice algorithms.
  • VisuAlgo - Interactive visualizations of data structures and algorithms.

2. System Design

  • System Design Primer (GitHub) - Comprehensive guide to system design.
  • Grokking the System Design Interview - Popular course on system design.
  • Gaurav Sen - YouTube Channel - Detailed explanations of system design concepts.

3. Mock Interviews

  • Interviewing.io - Practice mock interviews with engineers.
  • Pramp - Free peer-to-peer mock interview platform.
  • LeetCode Mock Interviews - Simulate interview environments.

4. Books

  • Cracking the Coding Interview - By Gayle Laakmann McDowell.
  • The Algorithm Design Manual - By Steven Skiena.
  • Elements of Programming Interviews - By Adnan Aziz, et al.

5. Google-Specific Resources

  • Google Careers - How We Hire - Official guide to Google's hiring process.
  • YouTube - Google Coding Interview Example - Example problems solved in Google interviews.
  • Google Interview Experience - Insights from successful candidates.

6. Coding Practice Platforms

  • LeetCode - Practice coding questions categorized by difficulty.
  • HackerRank - Coding challenges and competitions.
  • Codeforces - Competitive programming platform.
  • CodeChef - Practice problems and monthly contests.
  • TopCoder - Practice, compete, and prepare for interviews.
